# :gift: InstallThemAll
Simple batch script to install the most common apps when you buy a new computer, setting up VM or etc...

## :loudspeaker: Installs:
- [WinRar (Activated)](https://win-rar.com)
- [SublimeText (Activated)](https://www.sublimetext.com)
- [Python3](https://www.python.org/downloads/windows/)
- [Firefox](https://www.mozilla.org/en-US/firefox/new/)
- [VLC](https://www.videolan.org/vlc/index.html)
